# NavLib for Kingpin (Reconstructed)

A reconstructed version of the original Kingpin navigation library (`navlib`), reverse engineered from the released SDK and original binaries.

The goal of this project is source-level compatibility with the original navigation system, including `.nav` file support and behavior matching the original implementation.

## Features

- Compatible with original Kingpin navigation files
- Recreated from original SDK and binary analysis
- Preserves original node, route and cell structures
- Verified against original object code and runtime behavior
- Builds with Visual C++ 6.0 compatible settings

## Status

- Navigation loading and saving working
- Original `.nav` files supported
- Core navigation functions reconstructed and verified
- Behavior closely matches the original `navlib`
